# Blind Sync 取扱説明書  
（相性計測対話システム）

---

## 1. 本システムについて

Blind Sync は、合コンやグループ飲み会などの場面で、  
参加者同士の相性を数値とビジュアルで表示し、  
会話のきっかけを生み出すことを目的とした  
**相性計測対話システム**である。

本システムは、Web ブラウザ上で動作し、  
特別なソフトウェアをインストールすることなく利用できる。

---

## 2. 利用に必要なもの

本システムを利用するために、以下を準備する。

- パソコン（Windows / macOS / Linux）
- Web ブラウザ（Google Chrome 推奨）
- Python（ローカルサーバ起動用）
- Git（ソースコード取得用）

---

## 3. インストール方法

### 3.1 ソースコードの取得

ターミナル（またはコマンドプロンプト）を開き、  
以下のコマンドを実行する。

```bash
git clone https://github.com/fyu1130/blind-sync.git
```

### 3.2 フォルダの移動

```bash
cd blind-sync
```

---

## 4. 起動方法

### 4.1 ローカルサーバの起動

本システムではデータを正しく読み込むため、  
必ずローカルサーバを起動してから利用する。

以下のコマンドを実行する。

```bash
python3 -m http.server 8000
```

### 4.2 ブラウザでのアクセス

Web ブラウザを起動し、  
以下の URL にアクセスする。

```
http://localhost:8000/entry.html
```

---

## 5. 使い方

### 5.1 参加者登録

- 参加者は **4 人**で利用する。
- 画面の案内に従い、以下の情報を入力する。
  - ニックネーム
  - MBTI
  - 好きなお酒
- 全員分の入力が完了すると、質問画面へ自動的に移動する。

### 5.2 質問への回答

- 表示される質問に対して、選択肢から回答する。
- 質問は全部で **4 問**出題される。
- 各参加者が順番にすべての質問へ回答する。
- 回答が完了すると、結果画面へ移動する。

### 5.3 結果の確認

- 参加者同士の相性が自動的に計算される。
- 相性は **パーセンテージ表示**で確認できる。
- 相性ランキングがゲージバー形式で表示される。
- ペアをクリックすると、組み合わせごとのアドバイスが表示される。

---

## 6. 注意事項

- 本システムは **ローカル環境専用**である。
- 起動時は必ずローカルサーバを使用すること。
- ページを更新すると、入力内容が残る場合がある。
- 再度利用する場合は、ブラウザのデータをクリアすることを推奨する。

---

## 7. 想定される利用シーン

- 合コンや飲み会のアイスブレイク
- 初対面同士のコミュニケーション促進
- グループワーク前の雰囲気づくり

---

## 8. まとめ

Blind Sync は、  
MBTI と価値観質問を組み合わせることで、  
参加者同士の関係性を可視化し、  
自然な会話を生み出すことを目的としたシステムである。

操作は簡単で、  
誰でも直感的に利用できる設計となっている。
